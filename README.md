# Smart Pet Food Dispenser 🐾

An IoT-based smart pet food dispenser built using Arduino Cloud for remote monitoring and food dispensing control.

## Features
- Remote food dispensing
- Real-time monitoring
- Mobile + desktop dashboard
- Arduino Cloud integration
- Automated feeding support

## Arduino Cloud Setup
1. Create a device in Arduino Cloud
2. Add cloud variables for dispenser control
3. Upload the Arduino sketch
4. Connect the dashboard widgets
5. Monitor and control remotely

## Tech Used
- Arduino / ESP32
- Arduino Cloud
- IoT Dashboard
- Sensors & Servo Motor

## Project
This repository contains the code and dashboard setup used for the physical smart pet food dispensing machine.
